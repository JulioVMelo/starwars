## Iniciando

Acesse a raiz do projeto e baixe as dependências utilizando o comando a seguir

```shell
  yarn
```

Inicie a aplicação utilizando o comando a seguir

```shell
  yarn start
```

por padrão o react disponibiliza localmente http://localhost:3000

## Deploy
